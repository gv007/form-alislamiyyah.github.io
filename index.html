Tujuan Ponpes Al-Islamiyyah Gunungjati

  • Berhati khusyu’ nan ta’at

Program Unggulan Ponpes Al-Islamiyyah Gunungjati

Program Tahsin

Program yang dikhususkan untuk para santri guna memperbaiki bacaan Al Qur'an.

Program Tahfidz

Program untuk para santri guna menambah dan menjaga hafalan Al Qur'an Juz 30.

Program Qiroatul Kitab

Program yang dikhususkan untuk santri guna memperoleh wawasan keagamaan dari sumber otoritatif.
